# Data Engineer Intern Test - ETL Pipeline

This project demonstrates a simple ETL (Extract, Transform, Load) pipeline using Python.

## Project Overview
1. **Extract**: Retrieve cryptocurrency data from CoinGecko API.
2. **Transform**: Clean and format data using Pandas.
3. **Load**: Prepare data for RDBMS (PostgreSQL/MySQL).

## Tech Stack
- **Language**: Python 3.x
- **Libraries**: Requests, Pandas, SQLAlchemy
- **Database**: PostgreSQL (Relational Database)

## How to Run
1. Install requirements: `pip install requests pandas sqlalchemy`
2. Run the script: `python main.py`
